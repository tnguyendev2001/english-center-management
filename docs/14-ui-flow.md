# 14 - UI Flow

## Daily dashboard flow

Admin opens Dashboard:
- See today's classes.
- Click "Điểm danh" on a class.
- Attendance page opens with student list.
- Admin marks Present/Absent/Excused.
- Save.
- Return to Dashboard.
- Class now shows "Đã điểm danh".

## Payment flow

Admin opens Debt page:
- See students with remainingAmount > 0.
- Click "Thu tiền".
- Payment modal opens.
- Admin enters amount, date, method, note.
- Save.
- Invoice and debt are recalculated.
